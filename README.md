# GRAND ARCHIVE

a content library web-application

## Available Scripts

In the project directory run:

### `mongod`

to start a database

---

In the project/server directory run:

### `nodemon server.js`

to start a server

---

In the project directory run:

### `npm start`

to start a client

---

In the project directory run:

### `lt --port 3000`

to assign a unique publicly accessible url that will proxy all requests to locally running webserver,
needed for sharing
